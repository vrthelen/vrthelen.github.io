Let's go through the processes used in this project. Firstly there is the loadNextImage(breedID) method. This tries to use axios to get an API using the inserted API key. A response variable is set the the API text for the specific breed. This finds all the information we want to format for the project such as the cat's origin, temperament, and custom variables I decided to display. The method also grabs an array of photos for each breed. This is taken by the slideshow() method to go through that array when the button is pressed by the user. This method sets the image to be whatever i is within the array of images, then increments it. If i becomes non-existent within the array (too high) then it is reset.
Next, let's examine the structure of the project. Everything is contained within a Vue component, that creates a cat object with a set template. This is the titular Cat Component. It has a property called catname that is a string. This will be the breed ID and determines what cat is loaded up. The template displays all the information that was set up within loadNextImage and slideshow using several divs that display flex to align that content properly. Certain variables with scores are displayed using a styled div taking in the score as a width as was recommended in the instructions.
